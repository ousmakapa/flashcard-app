AI card generation

Import from PDF

Upload a PDF and OpenAI will generate Easy / Medium / Hard flashcards automatically. Needs internet and an OpenAI API key (a few US cents per PDF).

1
Get an API key — Go to Settings, paste your OpenAI key and save. Get a free key at platform.openai.com → API keys. Your key stays on this device only.
2
Choose a PDF & deck — Must be a text-based PDF (typed lecture notes, slides). Scanned image-only PDFs won't work. Pick an existing deck or type a new deck name below.
3
Review & add cards — Check the generated cards, use the Easy / Medium / Hard filters to focus, un-tick any you don't want, then click Add to save them.
